# binance4j-websocket

Binance4j-websocket is a Java library providing an API for interacting with the Binance websocket endpoints.

## Installation

### Maven

```xml
<dependency>
  <groupId>com.binance4j</groupId>
  <artifactId>binance4j-websocket</artifactId>
  <version>1.0.1</version>
</dependency>
```
